# LIbraryy
Разработка приложения для библиотеки 
  формы:
  
  form1- форма авторизации;
 
  form2- форма каталога книг; 
 
  form3- форма детальная информация о книге;
 
  form4- форма личный профиль;
 
  form5- форма меню библиотекаря;
 
  form6- форма управление каталога;
 
  form7- форма добавление/редактирование книг;
 
  form8- форма управления экземплярами;
 
  form9- форма возврат и выдачи книг;
  
  form10- форма журнал выдачей.

пользовательский элемент:

  usercontrol1- карточка книги для читателя;
  
  usercontrol2- карточка книги для библиотекаря;
  
  usercontrol3- карточка книга на руках.
